# Architecture Overview

```text
Telegram user
  -> command router
  -> tier/portfolio/quota check
  -> cached report path OR premium AI workflow
  -> markdown report
  -> Telegram delivery
```

## Free Path

Free users only read cached reports generated by the owner run.

## Premium Path

Premium users may upload a portfolio. Premium AI jobs are quota-limited:

- total: 8/day, 30/week
- debate: 1/day, 5/week
- scan: 1/day, 5/week
- options/IPO/bottleneck: 2/day each, 6/week each

## Safety

- no live trading
- no credential handling
- no portfolio sharing between users
- explicit example-report labeling for cached free output
